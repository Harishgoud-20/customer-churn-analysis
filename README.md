# customer-churn-analysis
Customer churn analysis project using MySQL and Power BI to identify high-risk customers and key churn factors such as tenure, contract type, and payment behavior.

# Project Overview
Analyzed 7,043 telecom customers using MySql to identify churn risk patterns and built an interactive Power BI dashboard

# Tools Used
MySql Workbench
Power BI Desktop
GitHub

# Project structure
Dataset  - Raw data
Sql      - SQL scripts
Power BI - Dashboard files

# Key Insights
Customers with month-to-month contracts churn more
Higher tenure = lower churn rate
Electric check payments show higher churn
